Construir um programa composto por 5 classes :

No (ou Nodo ) : Elemento construtivo das estruturas de dados. 
Deve armazenar um valor inteiro e também uma referência (ponteiro / apontamento / endereço ) para o próximo e o anterior elementos da estrutura.

Pilha : classe que implementa as estratégias de inserção e remoção LIFO.

Fila : classe que  implementa as estratégias de inserção e remoção FIFO.

Lista : classe que implementa as estratégias e inserção e remoção de listas duplamente encadeadas.

Classe principal : o programa com menu e interação com usuário, para testar as estruturas.
